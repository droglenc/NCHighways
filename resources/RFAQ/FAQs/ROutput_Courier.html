How do I make R output look nice without having to physically type in spaces or tabs?

R uses Courier New as its output font. Courier New is a constant spacing (rather than a proportional spacing) font which means that each character uses the same amount of horizontal space regarless of whether it is a "skinny" letter like "i" or a "fat" letter like "o". When pasting output from R into MS Word the output will use the font being used in MS Word -- usually NOT Courier New. In other words, your R output may look like the following; which is largely unreadable.

n NAs Valid n Mean St. Dev. Min. 1st Qu. Median 3rd Qu. Max.

150.00 0.00 150.00 5.84 0.83 4.30 5.10 5.80 6.40 7.90

This output can be made much more readable by pasting it into MS Word, highlighting the output in MS Word, and then changing the font to Courier New. The font can be changed with the drop-down box in the "Font" tab of the MS Word tool bar or by right-clicking on the highlighted text and selecting the "font" option. The pasted output in Courier New font is shown below in a much more readable format.

       n      NAs  Valid n     Mean St. Dev.     Min.  1st Qu.   Median  3rd Qu.     Max. 
  150.00     0.00   150.00     5.84     0.83     4.30     5.10     5.80     6.40     7.90

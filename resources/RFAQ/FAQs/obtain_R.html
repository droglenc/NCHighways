How do I obtain R for my home computer?

R is a free open-source program so you can download it and install it on your home computer free-of-charge. Follow these steps to download R for a Windows machine

  1. Save this file to your hard drive (this file is ~30 mb so it may take a while to download).
  2. Locate the downloaded file on your hard drive (it will have a name similar to "R-2.X.X-win.exe".
  3. Run (i.e., double-click) this file.
  4. To use R with TINN-R then follow these steps (if not then install with all defaults (click "Next" at each option).).
    1. Use all the defaults until you see a dialog box that says "Select Components" change to "32-bit User Installation".


    2. In the dialog box that asks "Do you want to customize the startup options" select "Yes (customized startup)" and click "Next".


    3. In the next dialog box select "SDI (separate windows)" and click "Next".


    4. Use the default values in all subsequent dialog boxes.

You can then run R by double-clicking the R icon on the desktop.

If you are using R for statistics at Northland you will want to download and install the NCStats package (see this FAQ). You may also want (you will need to if using NCStats) to install other packages (see this FAQ). Finally, if you are using TINN-R then you will want to obtain TINN-R (see this FAQ) configure it to work with R (see this FAQ).

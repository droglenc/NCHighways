R can't find droplevel().

As of version 0.0-10 I removed the droplevel() from NCStats. The functionality of droplevel() is exactly replicated with drop.levels() of the gdata package. Thus, you can use drop.levels() in exactly the same manner as you used droplevel() (but make sure that the gdata package is loaded).

Alternatively, consider using Subset() from NCStats.

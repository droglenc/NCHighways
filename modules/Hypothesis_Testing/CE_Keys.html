Identifying Type I and II Errors

  1. The \(H_{0}: \mu=15.4\) and \(H_{A}: \mu>15.4\). A Type I error would be saying that the chicken is contaminated when it really is not contaminated. A Type II error would be saying that the chicken is not contaminated when it really is contaminated. In my mind, a Type I error is more egregious as people would be eating contaminated chicken that they have been told is not contaminated.
  2. The \(H_{0}\) is that the drug does not exceed specification whereas the \(H_{A}\) is that the drug does exceed the specification. A Type I error occurs if the drug does not exceed the specifications but it concluded that it does. A Type II error occurs if the the drug does exceed the specifications but it is concluded that it does not. In my mind, a Type II error is more egregious as people would be using a “bad” drug but would have been told that it was OK.

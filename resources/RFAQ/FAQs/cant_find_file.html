The read.table() does not find my data file.

This error typically looks like this ...

> d1 <- read.table("Ex1_spaces2.txt",header=T)
Error in file(file, "r") : unable to open connection
In addition: Warning message:
cannot open file 'Ex1_spaces2.txt', reason 'No such file or directory' in: file(file, "r")

The most common reasons for this error are (prioritized by frequency of occurrence) ...

  1. You have misspelled the actual name of the file.

  2. You have not changed the R working directory to where the data file is actually saved. The working directory can be set by choosing the "File" menu, "Change Dir ..." submenu, and the using the "Browse" button to browse to the directory or folder where you saved the file. Note that you won't actually see your file when you perform this operation, you just have to select the folder in which the file resides (i.e., you have to know where this folder is). This is described in Section 21.3.7 of the class text.

  3. You have not saved the file as a text file. Most commonly you saved it as an Excel file and should use read.xls() as described in this FAQ. Converting an Excel file to a tab-delimited text file is also described in this FAQ. Alternatively, you may have saved the file as a "comma-separated values" file and should be using read.csv().

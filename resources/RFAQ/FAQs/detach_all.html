Is it possible to detach all data frames at once?

You may want to detach all data frames if you have accidentally attached many data frames or if you just want to make sure that you have all data frames detached after finishing an analysis or before starting an new analysis. All currently attached data frames can be detached by typing detachAll() in R (note that the parentheses are needed and that the NCStats package must be loaded). For example, the following command shows that the d2 and d1 data frames are attached

> attached()
[1] "d2" "d1"
and the following detaches these two data frames 
> detachAll()
> attached()
No data frames are currently attached.

If you cannot remember if you have any data frames attached then see this FAQ. If you want to detach just one data frame then see this FAQ.

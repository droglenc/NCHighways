I can't remember the names of objects that I have created.

There are two methods for listing all named objects in Rs memory -- (1) type ls() or (2) choose the "misc" menu and "List objects" submenu. The first method is illustrated below.

> ls()
 [1] "attached" "chi1"     "crap"     "d1"       "d2"       "i"       
 [7] "junk"     "tan"      "tan1"     "tbl"      "txt"

R can't find ...

Most of the time this error occurs because you typed the name of the function incorrectly -- e.g., levenetest rather than levene.test. If the problem is not a typo then this error typically occurs because you do not have a required package loaded. More often than not in this class that means that you do not have the NCStats package loaded. If the NCSTats package is installed then it can be loaded with ...

> library(NCStats)

If the NCStats package is not installed then see this FAQ.

Some functions that we will use are contained in other packages. Many of these packages are loaded with the NCStats package (thus, if you load NCStats you will not need to load these) but the specific functions and their packages are listed below. Thus, you can access these functions by installing and loading the corresponding functions.

FUNCTIONPACKAGE
ad.test()nortest
boxcox()MASS
combine()gdata
glht()multcomp

outlierTest()car

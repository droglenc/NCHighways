When I load the NCStats package R says that it cannot find some other packages.

The NCStats package is a package of routines that I have written to make some aspects of using R for statistics classes at Northland easier. This package requires these packages to work properly ...

  • car
  • gdata
  • gplots
  • Hmisc
  • nortest
  • plotrix
  • sciplot
  • TeachingDemos

If these packages are not properly installed then library(NCStats) will produce an error where R says it cannot find one of these packages.

Installing these other packages requires an internet connection. If you are using a Windows machine you can follow these steps to install these other packages.

  1. Open R
  2. Select the "Packages" menu and the "Install Packages" submenu.
  3. Select a mirror site in the U.S. from the ensuing dialog box.
  4. Select the packages listed above in the ensuing dialog box. Multiple packages can be selected at once by holding down the CTRL key while selecting from the list.
  5. Press the OK button.

If you are using a Macintosh you will need to install these packages through the Package Manager.

R will then take several minutes to download the required packages and install them.

How do I make a, m, or s or other Greek Letters?

Most Greek letters can be created by changing a "regular" letter to the Symbol font. For example, a is constructed by changing the letter "a" to Symbol font. Changing fonts in MSWord is accomplished by typing the equivalent letter in MS Word, highlighting the letter, and then changing the font to Symbol. The font can be changed with the drop-down box in the "Font" tab of the MS Word tool bar or by right-clicking on the highlighted letter and selecting the "font" option. The following table shows equivalent letters for common statistical symbols.

The last option above can be used to construct "x-bar." To do this type the symbol above (on the key above the Tab key) followed by an "x". Then highlight the symbol above and change the font to Symbol. This will create a character that looks like an "x-bar.".

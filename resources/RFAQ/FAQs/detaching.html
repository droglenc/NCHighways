How do I detach an attached data frame?

An attached data framed can be "un-attached" by placing the name of the data frame in detach(). For example

> str(d1)
'data.frame':   3 obs. of  4 variables:
 $ classes : int  3 4 1
 $ name    : Factor w/ 3 levels "Derek Ogle","Jefry Johnson",..: 1 3 2
 $ years   : int  11 19 3
 $ hometown: Factor w/ 3 levels "Ashland","Duluth",..: 3 2 1
> attach(d1)
> years
[1] 11 19  3
> detach(d1)
> years
Error: object "years" not found

If you cannot remember which data frames you have attached then see this FAQ. If you want to detach all currently attached data frames then see this FAQ.

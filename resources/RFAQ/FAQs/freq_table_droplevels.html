My frequency table contains unwanted levels.

This problem usually occurs when you are constructing a frequency table on a factor variable in a data frame created from subset(). R, unfortunately, keeps the names of the levels in the original data frame rather than just the names of the levels in the new data frame. However, the unused levels in the new data frame can be dropped by using Subset() instead (note the captial "S"). For example, this problem is illustrated with the following commands ...

> library(gdata)
> tan <- read.table("Tanner.txt",head=T)
> tan1 <- subset(tan,type=="slick.board" | type=="standard")
> table(tan1$type)
   cod.sock slick.board    standard ZK.excluder 
          0           9           9           0

This problem is avoided with Subset() as shown below ...

> tan1 <- Subset(tan,type=="slick.board" | type=="standard")
> table(tan1$type)
  slick.board    standard 
            9           9

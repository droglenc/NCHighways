Is there a text editor for the Macintosh?

R for the Mac has a built in editor that apparently is quite useful. Other editors for the Macintosh that have been suggested on the R-help mailing list are listed below (and linked to a download site):

  • ESS for Emacs (also for Windows)
  • JGR (also for Windows)
  • Aquamacs for Emacs (specifically for Mac OSX)
  • TextMate
  • BBEdit

A long list of potential editors with R code highlighting can be found here.

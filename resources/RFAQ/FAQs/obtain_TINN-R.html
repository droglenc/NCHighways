How do I obtain TINN-R for my home computer?

TINN-R is a free open-source program so you can download it and install it on your home computer free-of-charge. Unfortunately TINN-R only works for Windows machines (see this FAQ for a Mac suggestion). A brief description of TINN-R can be found at the TINN-R homepage.

The very latest version of TINN-R can be downloaded from this SourceForge.net page (note the download icon is on the left-side of the page). Suggestions for setting up TINN-R can be found in this FAQ.

How do I simulate data for a two-way ANOVA using mrnorm()?

The general use of the mrnorm() function for simulating data for a one-way ANOVA given observed values of the sample size, mean, and standard deviation is illustrated in this FAQ (I strongly urge you to become familiar with this FAQ first). In this FAQ I will demonstrate a method for using mrnorm() to simulate data for a two-way ANOVA.

For instance, suppose that you have six groups of data with the following characteristics

        Inland Sites      Coastal Sites
       ---------------   ---------------
Group   n   mean    SD    n   mean    SD
weasel  4   13.1   4.2    3    8.6   3.6
fox     5   17.2   3.6    4   12.2   3.2
hare    3    8.9   3.4    4    3.4   3.1

The trick to entering this data as a two-way ANOVA is to enter the data separately for each level of one of the variables as if it were being entered for a one-way ANOVA and then combine these parts into one larger data frame. In this example, I will proceed by entering the data for the inland sites first followed by the coastal sites. Note that this closely follows the procedure shown in this FAQ.

> group <- c("Weasel","Fox","Hare")

> # Inland sites first
> n.i <- c(4,5,3)
> mn.i <- c(13.1,17.2,8.9)
> s.i <- c(4.2,3.6,3.4)
> newdf.i <- mrnorm(n=n.i,mean=mn.i,sd=s.i,grp.labels=group,var.label=c("Response","Group"))
> newdf.i
    Response  Group
1  14.578280 Weasel
2  11.961132 Weasel
3  17.899201 Weasel
4   7.961388 Weasel
5  20.744079    Fox
6  13.295645    Fox
7  20.829703    Fox
8  13.904535    Fox
9  17.226038    Fox
10 11.491722   Hare
11 10.158005   Hare
12  5.050273   Hare

> # Coastal sites second
> n.c <- c(3,4,4)
> mn.c <- c(8.6,12.2,3.4)
> s.c <- c(3.6,3.2,3.1)
> newdf.c <- mrnorm(n=n.c,mean=mn.c,sd=s.c,grp.labels=group,var.label=c("Response","Group"))
> newdf.c
     Response  Group
1   9.1484771 Weasel
2   4.7572352 Weasel

These data can then be combined into one larger data frame and viewed with, 

> newdf <- gdata::combine(newdf.i,newdf.c,names=c("inland","coastal"))

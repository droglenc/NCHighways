Situation 1

  1. The percentage of X<0.11 is 54.4.
  2. The percentage of X>-0.11 is 54.4.
  3. The percentage of -1.45<X<1.11 is 6.0.
  4. The X that has 33% of Xs smaller is -0.44.
  5. The X that has 25$ of Xs larger is 0.67.
  6. The most common 90% of Xs are between -1.64 and is 1.64.

R Appendix

library(NCStats)
( q1 <- distrib(0.11,mean=0,sd=1) )
( q2 <- distrib(-0.11,mean=0,sd=1,lower.tail=FALSE) )
( q3a <- distrib(-1.45,mean=0,sd=1) )
( q3ab <- distrib(-1.11,mean=0,sd=1) )
q3ab-q3a

( q4 <- distrib(0.33,mean=0,sd=1,type="q") )
( q5 <- distrib(0.25,mean=0,sd=1,type="q",lower.tail=FALSE) )
( q6a <- distrib(0.05,mean=0,sd=1,type="q") )
( q6b <- distrib(0.95,mean=0,sd=1,type="q") )

Situation 2

  1. The percentage of Y<75 is 79.8.
  2. The percentage of Y>63 is 87.8.
  3. The percentage of 62.3<Y<72.9 is 58.6.
  4. The Y that has 30% of Ys smaller is 66.85.
  5. The Y that has 40$ of Ys larger is 70.00.
  6. The most common 50% of Ys are between 65.95 and is 74.05.

R Appendix

library(NCStats)
